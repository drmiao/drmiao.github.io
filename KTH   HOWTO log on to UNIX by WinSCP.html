HOWTO log on to UNIX by WinSCP

Instructions about how attend to the private web page on a remote UNIX using WinSCP.

Start WinSCP by Srart/programs/WinSCP3/WinSCP

  1. Host name: sftp.sys.kth.se
  2. User name: Your KTH username
  3. Password: Private KTH password
  4. Click Login

First time logging in or when using an other computer than earlier a warning shows, accept the server key by clicking Yes.


Your web page is stored in the Unix server public_html.
Pull files to publish in to right hand public_html


To see the own web page use a browser and visit http://people.kth.se/~yourusername where yourusername is swapped for first part of Your kth.se email "text pre @kth.se"

For none WIKS-computer:

Windows

Two programs that can manage SFTP and can be used as above are WinSCP and FileZilla.

MacOS 9 or earlier

By example MacSFTP that can manage both SFTP and SCP.

Mac OS X

By example Fugu that can manage both SFTP and SCP. Cyberduck ought to work as well.
